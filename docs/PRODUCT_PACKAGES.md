Calculate your complete carbon footprint across all emission scopes with scientific precision:

  - Cat 4: Upstream transportation
  - Cat 5: Waste generated in operations
  - Cat 6: Business travel
  - Cat 7: Employee commuting
  - Cat 8: Upstream leased assets
  - Cat 9: Downstream transportation
  - Cat 10: Processing of sold products
  - Cat 11: Use of sold products
  - Cat 12: End-of-life treatment
  - Cat 13: Downstream leased assets
  - Cat 14: Franchises
  - Cat 15: Investments (PCAF methodology)

- 401+ verified emission factors from authoritative sources
