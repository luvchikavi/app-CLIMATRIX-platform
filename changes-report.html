Problem & Solution

Problems Found

  • When a supplier emission factor was provided (e.g., IEC 0.467, Dorad 0.417), only the market-based calculation was performed
  • The location-based calculation was skipped entirely due to an early return in the pipeline
  • Supplier name and supplier EF were extracted during Excel parsing but never saved to the database
  • CSV/PDF exports had no columns for dual reporting or supplier data

What Was Fixed

  • Both location-based and market-based results are now calculated per GHG Protocol Scope 2 Guidance
  • Supplier name and emission factor are persisted in the Activity model
  • Dual results (location_co2e_kg, market_co2e_kg) stored in the Emission model
  • CSV and PDF exports now include all 4 new columns
  • API responses include supplier data and dual results
GHG Protocol Scope 2 Guidance: Organizations must report Scope 2 emissions using both location-based (grid average) and market-based (supplier-specific) methods. The primary CO2e result uses the market-based value when a supplier EF is provided.
⚙

How Dual Calculation Works

Excel Import (supplier_name="IEC", supplier_ef=0.467, quantity=1,203,111 kWh) | v CalculationPipeline._calculate_supplier_ef() | +---> [NEW] Market-Based Calculation | 1,203,111 kWh x 0.467 (IEC supplier EF) = 561,852.84 kg CO2e | +---> [NEW] Location-Based Calculation | 1,203,111 kWh x 0.502 (Israel grid factor) = 603,961.72 kg CO2e | v Result co2e_kg = 561,852.84 (primary = market-based) market_co2e_kg = 561,852.84 [NEW FIELD] location_co2e_kg = 603,961.72 [NEW FIELD]

Where to See Results in the App

  1. CSV Report Export

    Go to Reports page (sidebar) → select the period from the dropdown (top right) → click the CSV button. The downloaded file includes 4 new columns: Supplier Name, Supplier EF, Location CO2e (kg), Market CO2e (kg).

    File: ghg_report_<period>.csv
  2. PDF Report Export

    Same page → click the PDF button. The PDF table includes 3 new columns: Supplier, Loc (t), Mkt (t) showing dual results in tonnes.

    File: ghg_report_<period>.pdf
  3. Activities API Response

    The GET /api/periods/{id}/activities endpoint now returns supplier_name, supplier_ef on each activity, and location_co2e_kg, market_co2e_kg on each emission.

    Endpoint: /api/periods/{id}/activities
  4. Reports → By Scope Tab

    On the Reports page, the By Scope tab shows Scope 2 activities with their primary CO2e value (market-based when supplier EF is provided). The dual breakdown is available in the CSV/PDF exports.

    Reports → By Scope → Scope 2
